[STEP4-1: Branch Class Sync (form-urlencoded)]
- Add POST /api/branch/{branch_id}/classes/sync
  - Accept application/x-www-form-urlencoded
  - Required fields: apt_id, classes_json
  - Optional: asof
  - Parse classes_json (JSON string) into list of classes
  - Upsert into new table branch_class_cache:
    (apt_id, branch_id, class_id) PK
    name, start, end, capacity, reserved, asof, updated_at
  - Return: {success:true, upserted:N}

- Add GET /api/branch/{branch_id}/classes?apt_id=...&date=...
  - Return cached list (optionally filter by date)

- Add minimal UI on Chat page:
  - If user asks "강좌 목록/강좌 리스트/오늘 강좌", call cache and render table.

- Provide clear error response if JSON parse fails:
  {success:false, error:"...", preview:"...", hint:"classes_json must be valid JSON array"}
